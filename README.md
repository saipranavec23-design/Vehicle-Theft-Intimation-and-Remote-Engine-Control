# Vehicle Theft Intimation and Remote Engine Control

## Overview
Vehicle Theft Intimation and Remote Engine Control is a smart vehicle security system designed to prevent vehicle theft. The system detects unauthorized access to a vehicle and immediately sends an alert notification to the vehicle owner. It also allows the owner to remotely disable or enable the engine using a mobile device.

This project combines embedded systems, communication modules, and sensors to provide real-time monitoring and control of a vehicle.

## Features
- Instant theft alert notification to the owner
- Remote engine shutdown capability
- Remote engine restart option
- Live vehicle monitoring
- Improved vehicle security system

## How It Works
1. The system is installed inside the vehicle and connected to the ignition system.
2. when ignition is detected, the system sends an alert to the owner's phone.
3. The owner can remotely send commands through a mobile application or message.
4. The ESP32 receives the command and controls the engine through a relay module.
5. The engine can be disabled or enabled remotely.

## Components Used
- Microcontroller- ESP32
- GSM Module (SIM800L)
- Relay Module
- Power Supply
- Mobile device for receiving alerts and sending commands
- GPS Module - NEO-6M

- Smart transportation security
